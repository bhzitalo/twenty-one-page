# 🎧 Twenty One Page |-/

Projeto desenvolvido com o objetivo de praticar e consolidar conhecimentos em **HTML, CSS e JavaScript**.

## 📌 Sobre o projeto

O **Twenty One Page** é uma página dedicada à banda *Twenty One Pilots*, como forma de aprendizado e experimentação em desenvolvimento web.

**O site apresenta:**

- Um breve resumo sobre a banda
- Um carrossel de imagens
- Uma seção com álbuns organizados em formato de cards
- Estrutura semântica com HTML5
- Estilização com CSS (incluindo Grid e responsividade básica)
- Interatividade com JavaScript

## 🎯 Objetivo

**Este projeto tem como foco:**

- Praticar estruturação de páginas com HTML
- Trabalhar layout e design com CSS
- Implementar interações com JavaScript
- Evoluir na organização e boas práticas de código

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## ⚠️ Observação

Este projeto não possui fins lucrativos e foi desenvolvido apenas para fins educacionais.

*Feito com 💛 por um fã de Twenty One Pilots*
